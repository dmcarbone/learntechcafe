Create-a-Free-Blog – Part 2#

Today we’re going to get hands-on and generate a blog locally using Hugo. This means that it won’t be live on the internet yet -that’s for Part 3! If you need a refresher on the what we’re trying to do, see Part 1. To recap, we will generate your intial blog website locally using Hugo, pick a theme from the Hugo website’s many free themes, install that theme, generate your site locally and view it in-browser! That’s right, you’ll get to see you site up and running locally before you know it.

Checklist#

Here’s a list of everything you’ll need in today’s lesson. Don’t worry, we’ll go over each in more detail.

  • A computer with a stable internet connecion (Windows, Mac or Linux – the process is the same)
  • A command line tool for a few basic commands.
  • A code editor (recommend using Visual Code Studio)

What is Hugo?#

Hugo is a free, open-source static site generator written in Go (a programming language, you don’t need to worry about the details for now). It generates you a site based on a preset-theme, and you as the user can create content for that site using Markdown, a simple foramtting language. A static site generator is a tool that builds web pages; it compiles content into static files that can be hosted anywhere – making your site faster, more secure, and easier to maintain compared to xxx.

Install Hugo and Git#

I’m going to breakdown how to install Hugo for the big three operating systems. If you need more detail than I provide here, please see the Hugo website for instalation instruction .Let’s open up your command line tool and install Hugo. All operating systems have one pre-installed. You’ll also need to install Git, which allows the installaion of various themese for Hugo.

Windows#

  1. Install a package manager. I recommend Scoop, an open-source package manager. In Windows, search for PowerShell, right click and Run as Adminstrator. Type the following command and press Enter:
Set-ExecutionPolicy RemoteSigned -scope CurrentUser

When prompted, type Y and press Enter. 2. Install Scoop using the same method, type this in a press Enter:

irm get.scoop.sh | iex
  1. Install Hugo by typing the folowing, then press Enter. We are installing the extended version of Hugo, which provide extra featres that some themes need.
scoop install hugo-extended
  1. Now, verify that hugo installed correctly. Run:
hugo version

You should see something like this returned in the terminal:

hugo v0.xx.x-XXXXXXX+extended windows/amd64
  1. You must also install Git for Windows. This is required for theme installation. Follow these instructions, just click the first ‘Click here to donwload’ link under the title ‘Download for Windows’. Open the downloaded file and follow the installation instrctions to install Git.

macOS#

  1. Install a package manager. Homebrew is the most common package manager on macOS. If you don’t have it installed already, first, open your terminal and paste the following command.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Follow the instructions on-screen, it may ask for your Mac password. This will install Homebrew for you. 2. Once Homebrew is installed, run:

brew install hugo

This installs the extended version of Hugo, which provides extra features that some themes require. 3. Install Hugo by typing the folowing, then press Enter. We are installing the extended version of Hugo, which provide extra featres that some themes need.

scoop install hugo-extended
  1. Now, check that Hugo installed correctly. Run:
hugo version

You should see something like this return in the terminal: hugo v0.xx.x+extended darwin/arm64 (or amd64) 5. You must also install Git. Run:

brew install git

Linux#

Note that these instructions are for Linux Mint 22.1 Cinnamon – instructions may very depending on the distruction.

  1. Download the latest Hugo extended version manually. Go to the official Hugo GitHub Releases page, find the latest version of the following and download it to your Downloads folder.
hugo_extended_0.xx.x_Linux-64bit.tar.gz
  1. Open the terminal and run the following. See Linux installing instructions for more details.
cd ~/Downloads
tar -xzf hugo_extended_0.xx.x_Linux-64bit.tar.gz

This unzips a hugo binary file. Move it to a directory in your PATH. In terminal, run:

sudo mv hugo /usr/local/bin/

Then, make it executable:

sudo chmod +x /usr/local/bin/hugo
  1. Confirm it works:
hugo version

You should see something like the following returned:

hugo v0.xx.x+extended linux/amd64
  1. Install Git for themes. In terminal, run:
sudo apt install git

That sums it up for how to Install Hugo and Git!

Install Visual Code Studio#

Now that Hugo and Git are installed, you should install a code editor if you don’t have one aleady. I recommend Visual Code Studio. This is pretty straightforward, simply download the file for your operating system and follow the installation instructions.

Select and Install a Theme for Your Blog#

Now for the exciting part! Go to the Hugo Themes website and explorer the available free themse. Click on several and explore the various styles of website your interested in. Each theme is curated by a open-source contributor - big should out to their hard work! In this tutorial, we will be using the PaperMod theme, which is clean and professional-looking. It just so happens that’s the theme we use here at LearnTechCafe!

I recommend using PaperMod so you can follow along with this tutorial more easily, however the same installation principles apply all themese.

Here’s how installation works. Each theme is maintained in a GitHub repository (repo). You have two options when using a one of theses themes:

  1. Connect to the live repo, and receive live updates as you build you website. We will not show this option in this tutorial.
  2. Download the theme from the live repo, and use the downloaded, static copy of the theme in your website. This is the option I recommend for beginners, since a static copy of the theme will never be changes, thus not breaking your website accidently.

To install a static copy of the PaperMod theme just go to the theme’s page on the Hugo website here and click Download. It will take you to the GitHub repo for that theme (PaperMod is maintained by Aditya Lange – thanks!). Select the green <> Code button and then Download ZIP.

EDIT insert image of repo page, highlight green code button

Go to your Downloads folder and unzip the file you just downloaded. Copy this unzipped folder. Open your local website folder you just created, in this case, myblog and open the themes folder. It should be empty. Paste your copied theme here. Done!

Create Your First Post#

Open your terminal. To generate your first post, run:

hugo new posts/my-first-post.md

Change my-first-post to any name you’d like. You can always change it later in the folders if you’d rather work manually with folders - check out myblog -> posts -> my-first-post.md. Just renamed it there! Or, create a new file there manually, just make sure it ends in the extension md. Open the post in any text editor to view the content, although I recommend opening it in VS Studio. Note that draft = false. Add one sentence below the final **—$$. It doesn’t matter what your write, just add some quick content. Save.

Run the Local Server#

Now let’s see what your new site actually looks like locally! Run this command in terminal:

hugo server

Visit http://localhost:1313/ in your browser.

Congradulations! You now have a basic blog running on your machine!
